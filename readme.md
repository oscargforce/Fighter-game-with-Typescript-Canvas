### Fighter game.  

https://oscargforce-fighting-game.netlify.app/

### Keybinds:
 - The Evil Wizard.
    - Movements: ```w, a, d```
    - Attack: ```s```

  - Medieval warrior
    - Movements: ```arrowUp, arrowLeft, arrowRight```
    - Attack: ```arrowDown```
